# Reviewer Traceability Index

## How to Use This Document

This index maps **every quantitative claim** in the manuscript to the **specific script, data file, and result file** that produced it. Reviewers can:

1. Find any claim in the paper (organized by section and line number)
2. Follow the link to the generating script
3. Run the script to reproduce the result
4. Inspect the intermediate data files

**Notation:**
- `→ script`: the Python script that generates this result
- `→ data`: the input data file(s) used
- `→ output`: the JSON/CSV result file where the number is stored
- `→ figure`: the figure file generated (if applicable)
- `[inline]`: result was computed in an interactive session (reproducible via the method described)
